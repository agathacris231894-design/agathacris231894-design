## 📌 Projetos

## 🧬 Iniiação Científica em Análise de Dados Genéticos e Construção de Árvores Filogenéticas - UFSCar Sorocaba (2023-2024)

Projeto de Iniciação Científica desenvolvido entre **2023 e 2024**, com o objetivo de processar, analisar e interpretar dados genéticos utilizando Python, com foco na construção de árvores filogenéticas para estudo de relações evolutivas entre espécies.

## 🚀 Objetivo da IC

Desenvolver uma solução computacional capaz de:
- Tratar e organizar dados genéticos (sequências de DNA/RNA)
- Realizar análises comparativas entre sequências
- Gerar árvores filogenéticas
- Auxiliar na interpretação de relações evolutivas

## 🧠 Conceitos Envolvidos
- Bioinformática  
- Filogenia  
- Alinhamento de sequências  
- Distância genética  
- Evolução molecular
